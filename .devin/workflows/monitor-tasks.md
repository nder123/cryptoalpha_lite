# Мониторинг задач из scratchpad.md

**Описание**: Автоматически проверяет файл scratchpad.md и выполняет задачи с "- [ ]"

**Шаги**:
1. Прочитать файл /home/ander/CascadeProjects/cryptoalpha_lite/docs/scratchpad.md
2. Найти все строки с "- [ ]" в разделе "## Задачи для параллельного выполнения"
3. Для каждой найденной задачи:
   - Если задача касается тестов или документации → передать в Codex CLI
   - Если задача сложная → выполнить самостоятельно
   - Если задача содержит слова "тест", "документац", "readme", "comment":
     - Выполнить команду: `codex --agent "задача" --context docs/scratchpad.md --yes`
4. После выполнения отметить задачу как "- [x]"
5. Добавить запись в "## Прогресс выполнения"
6. Повторять каждые 30 секунд (фоновый режим)
