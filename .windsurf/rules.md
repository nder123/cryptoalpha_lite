# Cascade Rules for CryptoAlpha Lite

## ОБЯЗАТЕЛЬНЫЕ ПРАВИЛА

1. **Перед каждым ответом** читай файл /home/ander/CascadeProjects/cryptoalpha_lite/docs/scratchpad.md
2. **После каждого действия** обновляй /home/ander/CascadeProjects/cryptoalpha_lite/docs/scratchpad.md
3. **scratchpad.md** — единственный источник правды о состоянии проекта

## Режимы работы

### Когда видишь "ЗАДАЧА:" в чате
- Прочитай scratchpad.md
- Спланируй выполнение
- Запиши план в scratchpad.md
- Запусти /home/ander/CascadeProjects/cryptoalpha_lite/windsurf_codex_bridge.sh
- Дождись выполнения
- Обнови scratchpad.md
- Напиши отчет

### Когда видишь новую задачу в scratchpad.md
- Автоматически начинай выполнение
- Не жди ручной команды

## Важно!
Если в корне проекта появляется файл .current_task — 
немедленно прочитай его и выполни указанную задачу.
После выполнения удали .current_task.
